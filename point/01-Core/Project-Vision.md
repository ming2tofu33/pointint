# Project Vision

> **Status:** Active
> **Last Updated:** 2026-03-27

## 1. 한 줄 정의

이미지를 나만의 Windows 커서로 만드는 가장 쉬운 제작 워크플로우

## 2. 브랜드 라인

- **Main:** `Your Point, Your Tint.`
- **Concept:** `Pointint for 모니테리어`
- **현재 제품:** 이미지를 나만의 Windows 커서로 가장 쉽게 바꾸는 방법

## 3. 비전

Pointint는 모니테리어를 위한 브랜드이자 제작 플랫폼이다. 다만 첫 제품은 넓게 시작하지 않고, **이미지를 나만의 Windows 커서로 만드는 가장 쉬운 워크플로우**에 집중한다.

Pointint는 커서 다운로드 사이트가 아니라, **내가 원하는 디지털 오브젝트를 직접 만들고 검증하는 제작 파이프라인**이다.

## 4. 해결하려는 문제

Pointint가 풀고 싶은 문제는 "커서 파일을 만든다"가 아니다.

- 디지털 공간을 내 취향대로 꾸미는 과정이 생각보다 불편하다
- 기존 커서 생태계는 다운로드 사이트, 설치형 편집기, 브라우저 확장으로 파편화되어 있다
- 이미지가 있어도 실제로 예쁘고 쓸 만한 커서로 만들기 어렵다
- Hotspot, 크기, 배경 대비, 알파 처리 문제로 결과물이 불편해지기 쉽다
- 적용 전에 느낌과 품질을 확인하기 어렵다

Pointint는 이 문제를 **제작 워크플로우**로 해결한다.

## 5. 핵심 원칙

- **브랜드는 넓게, 첫 제품은 좁게:** Pointint는 장기적으로 모니테리어 브랜드지만, 초반 제품은 커서 제작에 집중한다
- **MVP는 커서 제작 흐름 검증:** 업로드 기반 커서 제작이 실제로 다운로드와 적용까지 이어지는지 증명하는 것이 우선이다
- **무료 경험은 강하게:** 비회원도 충분히 만들고 써볼 수 있어야 한다
- **AI는 MVP 이후 단계적 도입:** 먼저 업로드 흐름을 증명하고, 이후 AI 보조 -> AI 생성 순서로 붙인다
- **장기 상품 단위는 테마팩:** 커서, 배경화면, 펫 같은 오브젝트를 하나의 모니테리어 패키지로 묶는 방향을 지향한다

## 6. 제품 포지셔닝

Pointint는 아래 셋 중 세 번째에 해당한다.

- 다운로드 사이트
- 단순 편집기
- **제작 워크플로우**

핵심 파이프라인:

`이미지 업로드 -> 편집 -> Hotspot 지정 -> 시뮬레이션 -> 다운로드`

## 7. V1이 성공한 상태

- 비회원이 이미지 한 장으로 커서를 만들고 다운로드까지 끊김 없이 완료한다
- 시뮬레이션에서 실제 사용감을 확인한 뒤 다운로드하는 비율이 의미 있게 높다
- 사용자가 만든 커서를 실제로 Windows에 적용한다
- 로그인 전환이 "저장하고 싶다"는 자연스러운 동기에서 발생한다
