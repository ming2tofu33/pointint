# Monetization Strategy

> **Status:** Active (Tint Economy 기반으로 재설계)
> **Last Updated:** 2026-03-27
> **Economy Design:** [[2026-03-27-pointint-economy-design]]

## 수익화 철학

강한 무료 제작 경험으로 유저를 데려오고, AI 크레딧(Tint)으로 첫 수익을 만들며, 공식 콘텐츠와 마켓플레이스로 생태계 수익을 확장한다. 월 구독은 사용하지 않는다.

## 수익원 4개

| # | 수익원 | 도입 시점 | 과금 방식 |
|---|---|---|---|
| 1 | **Tint 팩 판매** | Phase 3 | 일회성 팩 구매 (₩4,900~₩39,900) |
| 2 | **프리미엄 기능** | Phase 2~ | Tint 소모 (17종 자동 변형 3,000T, HiDPI 1,000T) |
| 3 | **공식 콘텐츠 판매** | Phase 5 | 개별 판매 (테마팩 ₩2,000~9,900) |
| 4 | **마켓 수수료** | Phase 5 | 거래당 20% 소각 |

## 경제 순환 구조

```
[실제 돈] → Tint 구매 → [유저 지갑]
                            │
              ┌─────────────┼─────────────┐
              ▼             ▼             ▼
          AI 사용        마켓 구매      콘텐츠 구매
          (소멸)           │          (Pointint 수익)
                     ┌─────┴─────┐
                  판매자 80%   수수료 20%
                  (이전)       (소각)
                     │
                 [판매자 지갑] → AI/콘텐츠에 재소비 → 더 좋은 창작 → 다시 판매
```

## Phase별 수익 로드맵

| Phase | 수익 요소 | 역할 |
|---|---|---|
| **1-2** | 없음 | 무료 도구로 유저 확보. 제작→다운로드 흐름 검증 |
| **3** | Tint 팩 판매 | 첫 수익. AI 보조 = Tint 소모. 출석 적립 시작 |
| **4** | Tint 소모처 확대 | AI 생성 추가. 프리미엄 내보내기 |
| **5** | 마켓 + 콘텐츠 | 순환 경제 완성. 본격 수익 |

## 전환 트리거

| 전환 | 트리거 | 핵심 질문 |
|---|---|---|
| Guest → Member | 저장/17종 욕구 | "이 커서를 잃고 싶지 않다" |
| Member → Tint 구매 | AI 사용 욕구 | "무료로 써봤는데 더 쓰고 싶다" |
| 구매 → 반복 구매 | Tint 소진 | "또 만들고 싶다" |
| Member → Creator | 활동 기반 자동 승급 | "내가 만든 걸 팔고 싶다" |

## Tint 소스 / 싱크 균형

### 소스 (유입)

| 소스 | 양 | 성격 |
|---|---|---|
| Tint 팩 구매 | 주요 유입 | 실제 매출 |
| 일일 출석 | +50/일 | 재방문 미끼 |
| 커서 제작 완료 (일 1회) | +50/일 | 제작 활동 유도 |
| 활동 보상 | 소량 | 참여 유도 |
| 친구 초대 | 일회성 보너스 | 바이럴 |

### 싱크 (소멸)

| 싱크 | 양 | 성격 |
|---|---|---|
| AI 보조 | 500 Tint/회 (확정) | 바닥 수요 |
| 17종 자동 변형 | 3,000 Tint (확정) | 핵심 프리미엄 |
| AI 생성 | Phase 4에서 결정 | 모델 비용 기반 |
| 공식 콘텐츠 구매 | 수천~수만 Tint | 감정적 가치 |
| 마켓 수수료 | 거래의 20% | 자동 소각 |
| 마켓 등록비 | 소량 | 스팸 방지 + 소각 |

**건강한 경제:** 싱크 ≥ 소스 → Tint 가치 유지

## 가격 원칙

- Guest는 약하면 안 된다 — 무료 제작 경험이 hook
- 첫 결제는 부담 없어야 한다 — Starter ₩4,900
- 너무 싸면 품질 의심 — Etsy 17종 ₩13,000~19,000 앵커 참고
- AI 생성 가격은 Phase 4에서 모델 비용 기반으로 결정
- 올리기는 쉽지만 내리기는 어렵다 — 보수적으로 시작
- 장기 핵심 상품은 테마팩이다
