# Acquisition Strategy

> **Status:** Active
> **Last Updated:** 2026-03-27
> **Purpose:** "유저를 어떻게 데려올 것인가?"에 대한 채널별 전략

---

## 1. 핵심 제약 조건

- **솔로 프로젝트:** 예산과 인력이 극도로 제한적
- **니치 시장:** "Windows 커서 제작"은 대중 키워드가 아님
- **바이럴 약점:** 커서는 다른 사람에게 보이지 않음 → 자연 확산이 어려움

따라서 전략은 **비용 0~저비용, 시각적 콘텐츠 주도, 인접 시장 공략**이어야 한다.

## 3. 채널별 상세

### 1순위: 숏폼 영상 — "와우 모먼트를 보여준다"

Pointint의 가장 강력한 마케팅 자산은 **"이미지가 커서로 변하는 과정"**이다. 이건 텍스트로 설명하는 것보다 30초 영상으로 보여주는 게 100배 강하다.

**심리 근거:** Availability Heuristic — 사람들은 직접 본 것을 더 쉽게 시도한다. "이런 게 가능하구나"를 보여주는 순간 시도 욕구가 생긴다.

#### 콘텐츠 공식

| 포맷 | 예시 | 기대 효과 |
|---|---|---|
| **변환 과정** | "내 고양이를 커서로 만들어봤다" (30초) | 와우 모먼트 → 공유 |
| **비포/애프터** | 기본 커서 → Pointint 커서, 데스크톱 전체 변신 | 시각적 대비 |
| **AI 생성 시연** | "텍스트 한 줄로 커서 만들기" (Phase 3~) | AI가 시각적으로 강력 |
| **데스크톱 셋업** | "내 데스크톱 꾸미기 마지막 터치 = 커서" | 인접 커뮤니티 연결 |
| **밈/트렌드 활용** | 유행 캐릭터/밈을 커서로 제작 | 트렌드 편승 |

#### 실행

- Phase 1 MVP만으로 콘텐츠 제작 가능 (AI 불필요)
- 주 2~3개 숏폼 업로드
- 해시태그: #desktopsetup #customcursor #커서꾸미기 #데스크톱꾸미기
- 비용: 0 (화면 녹화 + 편집)

---

### 2순위: 인접 커뮤니티 — "커서 커뮤니티가 아니라 꾸미기 커뮤니티"

커서 전용 커뮤니티는 작다. 하지만 **데스크톱 커스터마이징 커뮤니티**는 크다.

| 커뮤니티 | 규모 | Pointint 포지셔닝 |
|---|---|---|
| **r/Rainmeter** | 구독자 50만+ | "데스크톱 셋업의 마지막 터치" |
| **r/desktops** | 구독자 10만+ | 셋업 공유 시 커서도 함께 |
| **r/unixporn** | 구독자 100만+ (참고) | 리눅스지만 커스터마이징 문화 참고 |
| **데스크테리어 카페/커뮤니티** | 한국 시장 | 모니테리어 개념과 직접 연결 |
| **Wallpaper Engine 커뮤니티** | Steam 수백만 유저 | 배경화면 다음 = 커서 |

#### 접근법

Reddit 자기홍보는 역효과. 대신:

1. **먼저 커뮤니티 멤버로 참여** — 다른 사람 셋업에 댓글, 도움, 피드백
2. **자기 데스크톱 셋업을 공유** — 그 안에 Pointint 커서가 자연스럽게 포함
3. **"어떤 커서 쓰세요?" 질문에 답변** — "직접 만들었어요, Pointint라는 사이트에서"
4. **절대 링크 먼저 던지지 않는다** — 물어보면 그때 알려주기

이건 수개월 걸리는 장기 전략이지만, 신뢰 기반으로 쌓이면 가장 강력하다.

### 3순위: SEO — "검색 의도에 맞는 페이지"

#### 주의: 검색 의도와 제품의 갭

- "how to change cursor windows" → 기존 파일로 바꾸는 법을 찾는 사람
- "custom cursor maker" → 이게 Pointint와 의도가 일치하는 키워드

**잘못된 SEO:** "커서 바꾸는 법" 가이드를 쓰고 Pointint로 유도 → 의도 불일치로 이탈
**올바른 SEO:** "내 이미지로 커서 만들기", "커서 직접 만드는 사이트" → 의도 일치

#### 키워드 전략

| 키워드 유형 | 예시 | 의도 일치도 |
|---|---|---|
| **직접 제작 키워드** | "custom cursor maker", "make your own cursor", "커서 만들기 사이트" | 높음 |
| **캐릭터/테마 키워드** | "anime cursor maker", "pixel cursor generator", "고양이 커서" | 높음 |
| **적용 가이드 (유입 후 전환)** | "windows 커서 바꾸는 법" → 가이드 제공 + "직접 만들어보세요" CTA | 중간 |
| **데스크톱 꾸미기 롱테일** | "데스크톱 꾸미기", "desktop customization guide" | 중간 (넓음) |

#### 프로그래매틱 SEO (Phase 3~)

유저가 만든 커서가 갤러리에 쌓이면:
- 각 커서 페이지가 검색에 인덱싱
- "cat cursor download", "anime cursor set" 같은 롱테일 자동 생성
- **유저 콘텐츠 = SEO 자산**

### 4순위: Product Hunt — "일회성 스파이크, 기대치 낮추기"

- Phase 1 MVP 완성 후 1회 런칭
- PH 유저는 주로 개발자/테크 → Pointint 1순위 타깃(일반 사무직)과 불일치
- 기대: **백링크 확보 + SEO 도메인 신뢰도 상승** (트래픽보다 이게 진짜 가치)
- "The easiest way to turn your image into a Windows cursor"

---

### 5순위: 크리에이터 파트너십 (Phase 3~)

AI 기능이 추가되면 마케팅 소재가 풍부해진다.

| 파트너 | 형태 | 기대 효과 |
|---|---|---|
| **데스크톱 셋업 유튜버** | "커서까지 바꿔봤다" 영상 제작 | 타깃 오디언스 직접 도달 |
| **일러스트레이터/캐릭터 IP** | 공식 커서 팩 협업 제작 | 팬덤 유입 + Tint 소비 |
| **주변기기 브랜드 (Razer, Logitech)** | 브랜드 커서 팩 | 인지도 + 공식 콘텐츠 |
| **Wallpaper Engine** | 교차 프로모션 | 인접 시장 유저 획득 |

---

### 6순위: 마켓 기반 자기 확장 (Phase 5~)

마켓이 열리면 **크리에이터 = 마케터**가 될 수 있다. 다만 닭-달걀 문제를 먼저 해결해야 한다.

#### 닭-달걀 부트스트랩 전략

| 단계 | 액션 |
|---|---|
| **1. 공급 먼저** | Pointint가 직접 공식 커서 팩 10~20개를 만들어 마켓에 채움 |
| **2. 초기 크리에이터 초대** | DeviantArt/커서 커뮤니티의 기존 크리에이터를 직접 초대 + 보너스 Tint 지급 |
| **3. 수요 연결** | 갤러리 → 마켓 자연 전환. "이 커서 마음에 드세요? 크리에이터의 다른 작품도 보세요" |
| **4. 자기 확장** | 크리에이터가 SNS에 자기 커서 홍보 → Pointint 유입 |

#### 바이럴 장치

| 장치 | 메커니즘 |
|---|---|
| **커서 쇼케이스 공유** | 만든 커서의 스크린샷/GIF를 소셜에 공유 (커서 자체에 워터마크 불가 → 쇼케이스 이미지에 "Made with Pointint" 삽입) |
| **데스크톱 스크린샷 챌린지** | #MyPointint 해시태그로 데스크톱 전체 셋업 공유 |
| **친구 초대** | 초대자/피초대자 각 2,000 Tint |
| **시즌 이벤트/드롭** | 한정 테마팩 → FOMO → 소셜 공유 |

## 5. North Star Metric

| Phase | North Star | 이유 |
|---|---|---|
| 1-2 | **제작 완료율** (시작 → 다운로드) | 흐름이 끊기면 아무것도 안 됨 |
| 3 | **Member → Tint 구매 전환율** | 수익 모델 증명 |
| 4 | **월간 AI 생성 횟수** | AI 가치 증명 |
| 5 | **마켓 월간 거래 수** | 생태계 증명 |

## 6. 하지 않을 것

| 채널 | 이유 |
|---|---|
| **디스플레이 광고 (무료 유저 대상)** | 브랜드 철학("기능보다 감각")과 충돌 |
| **유료 광고 (Phase 1-2)** | 제품 검증 전 광고 예산 투입은 낭비 |
| **인플루언서 대량 시딩** | 니치 제품에 대량 시딩은 비효율. 정밀 타깃만 |
