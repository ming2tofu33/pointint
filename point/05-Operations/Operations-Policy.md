# Operations Policy

> **Status:** Active
> **Last Updated:** 2026-03-27
> **Source:** `docs/운영 정책.md`

## 사용량 정책

- Guest: 제작 횟수 제한 없음. 일반 사용자가 자유롭게 쓰는 데 제한 없다
- 봇과 악용 사용자만 차단: Rate limiting, 파일 크기/해상도 제한, 동일 이미지 캐싱, 봇 탐지
- 정상 사용자는 제한을 느끼면 안 된다

## 파일 관리

- 비회원: 임시 저장 후 자동 만료, 서버에 보관하지 않음
- 로그인: 프로젝트 단위로 저장, 재수정 가능
- 저장/재수정은 로그인의 자연스러운 보상이다

## 비용 방어

| 방법 | 설명 |
|---|---|
| 해상도 정규화 | 처리 전 적정 크기로 리사이즈 |
| 결과 캐싱 | 동일 이미지 해시 기반 캐싱 |
| 모델 선택 | 비용 대비 품질 최적 |
| 클라이언트 처리 검토 | 가능한 것은 클라이언트에서 |

## 신뢰 & 안전

- 기본 적용: zip + .cur/.ani + 적용 가이드 (사용자가 직접 적용)
- 자동 설치(.inf)는 후순위
- 원복 가이드 항상 함께 안내

## 운영 모니터링

| 지표 | 목적 |
|---|---|
| 제작 완료율 | 업로드→다운로드 끊기지 않는 비율 |
| 시뮬→다운로드 전환율 | 시뮬레이션 가치 확인 |

| 배경 제거 처리 시간 | 대기 경험 |
| 서버 비용 / 건수 | 건당 비용 추적 |
