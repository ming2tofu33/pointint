# Cursor Editor

> **Status:** Active
> **Last Updated:** 2026-03-27
> **Source:** `docs/기능 상세.md` §2, §2-B

## 정적 커서 편집기 (.cur)

### MVP 조작 범위

| 조작 | 설명 |
|---|---|
| 위치 조정 | 커서 프레임(캔버스) 안에서 이미지를 이동 |
| 크기 조정 | 커서 규격에 맞춰 리사이즈 (16~128px) |
| Hotspot 지정 | 클릭 지점을 드래그로 지정. 사용성의 핵심 |
| 회전/뒤집기 | Rotate(90° step), Flip |

### MVP에서 제외

- 자르기(crop)
- 브러시/지우개 수동 보정
- 밝기/대비 등 색 보정

## 애니메이션 편집기 (.ani)

### 진입 경로

- **이미지 여러 장 업로드:** 각 이미지가 하나의 프레임이 된다
- **GIF 업로드:** 자동으로 프레임 분리

### MVP 조작 범위

| 조작 | 설명 |
|---|---|
| 프레임 순서 | 드래그로 프레임 순서 변경 |
| 프레임 속도 | 전체 또는 개별 프레임 재생 속도 조절 |
| 프레임 추가/삭제 | 프레임 단위 관리 |
| 애니메이션 미리보기 | 실시간 재생 확인 |

### 이후 고도화

- 프레임별 개별 편집 (위치, 크기, Hotspot 등)
- 프레임별 배경 제거
- 프레임 간 보간/전환 효과

## 핵심 원칙

- 편집기는 가볍게 유지한다
- "커서로 만드는 데 필요한 조작만" 있으면 된다
- 이미지 편집 앱이 아니라 커서 제작 도구다
- 애니메이션 MVP는 프레임 구성과 속도 조절에 집중
