# Market Validation Analysis

> [!info]
> **Status:** Working draft  
> **Last Updated:** 2026-03-27  
> **Basis:** `[[Market-Validation-Checklist]]` + 공개 웹 리서치

## Executive Summary

현재 공개 신호 기준으로 보면, **Pointint가 겨냥하는 시장은 존재한다.**  
다만 “커서 자체”만 보면 대형 범용 시장이라기보다 **작지만 꾸준한 니치**에 가깝고, 진짜 큰 기회는 **데스크톱 커스터마이징 전체**와 **모니테리어 확장성**에 있다.

핵심 결론은 아래와 같다.

1. **수요는 실재한다.** 브라우저 커서 확장, Windows 커서 앱, 데스크톱 커스터마이징 커뮤니티, Etsy 판매 데이터를 보면 “사람들이 커서를 꾸미고 싶어 한다”는 신호는 충분하다.
2. **하지만 시장은 파편화되어 있다.** 기존 강자는 대부분 다운로드 라이브러리, 브라우저 확장, 설치형 앱이다.
3. **Pointint의 기회는 제작 레이어다.** “내 이미지를 Windows 커서로 만드는 웹 기반 제작 흐름”은 아직 선명한 대표자가 약하다.
4. **돈은 콘텐츠와 편의에서 난다.** 단순 파일 다운로드보다, 17종 완성도, AI 보조, 공식 테마팩, 데스크톱 펫 같은 확장 상품이 더 유망하다.
5. **시장 검증의 1차 질문은 ‘커서에 돈을 쓰는가’가 아니라 ‘내 이미지를 커서로 만들고 싶어 하는가’다.** Pointint는 여기서부터 시작하는 것이 맞다.

---

## 1. 조사 방법과 한계

이번 리서치는 아래 공개 신호를 중심으로 했다.

- 브라우저 확장 사용자 수
- 공식/비공식 커뮤니티 규모
- Steam 리뷰와 팔로워 수
- GitHub stars
- Etsy 판매 상품과 리뷰
- 데스크톱 펫/커스터마이징 관련 기사와 상용 사례

### 한계

- Google Keyword Planner, Ahrefs, SimilarWeb 유료 계정 수준의 정량 검색량/트래픽 데이터는 직접 확보하지 못했다
- 따라서 이번 문서는 **정확한 SEO 수치 분석**보다는 **시장 존재 여부와 방향성 판단**에 더 강하다

즉, 이번 분석은 **“이 카테고리가 살아 있는가?”**를 보는 1차 검증으로 이해하는 것이 적절하다.

## 2. 강한 시장 신호

### 2.1 커서 커스터마이징 수요는 이미 존재한다

브라우저 커서 확장만 봐도 카테고리 자체가 유효하다는 신호가 크다.

- **Sweezy Cursors**는 Chrome Web Store에서 **300,000 users**, **1.6K ratings**, **4.8 rating** 신호가 보인다.  
  Source: [Chrome Web Store - Sweezy Cursors](https://chromewebstore.google.com/detail/sweezy-cursors-%E2%98%85-%E0%B9%80%E0%B8%84%E0%B8%AD%E0%B8%A3%E0%B9%8C%E0%B9%80%E0%B8%8B%E0%B8%AD/gdcfjidbchfpojnfifkgghbamkdmbdaf?hl=th)
- **Shimeji Browser Extension**은 Chrome Web Store에서 **1,000,000 users** 신호가 보이며, 페이지 자체에서도 **2.7K ratings**와 데스크톱 버디/캐릭터 상호작용을 강조한다.  
  Source: [Chrome Web Store - Shimeji Browser Extension](https://chromewebstore.google.com/detail/shimeji-browser-extension/gohjpllcolmccldfdggmamodembldgpc)
- **Custom Cursor for Windows** 페이지는 **“Over 8M users trust Custom Cursor”**를 전면에 내세운다.  
  Source: [Custom Cursor for Windows](https://custom-cursor.com/en/products/custom-cursor-for-windows)
- **Custom Cursor Community**는 공개 커뮤니티 첫 화면에서 **General Discussion 25,002**, **Cursor Ideas 6,458**, **Cursor Collections 202** 같은 주제 수를 보여준다.  
  Source: [Custom Cursor Community](https://community.custom-cursor.com/)
- **Cursors-4U**는 FAQ와 홈에서 **20년 이상 운영**과 **“Millions of downloads later”**를 강조한다.  
  Sources: [Cursors-4U FAQ](https://www.cursors-4u.com/faq), [Cursors-4U Home](https://www.cursors-4u.com/)

이 조합은 “커서 꾸미기”가 장난성 취미에 그치지 않고, 적어도 수십만~수백만 사용자 규모의 지속 수요를 가진 카테고리라는 점을 보여준다.

### 2.2 커서보다 더 큰 인접 시장은 충분히 크다

Pointint의 장기 방향이 `모니테리어`라면, 커서만 볼 게 아니라 데스크톱 커스터마이징 전체를 봐야 한다.

- **Wallpaper Engine**은 Steam에서 **222,652 English reviews**, **900,250 total reviews**, **$4.99 가격**, **over a million free wallpapers from the Steam Workshop**를 보여준다.  
  Source: [Wallpaper Engine on Steam](https://store.steampowered.com/app/431960/Wallpaper_Engine/)
- 같은 제품의 Steam DLC 페이지에는 **1,734,207 followers**가 보인다.  
  Source: [Wallpaper Engine DLC Page](https://store.steampowered.com/dlc/431960/Wallpaper_Engine/%3Fsnr%3D1_5_9__1061)
- **Lively Wallpaper**는 GitHub에서 **18.5k stars**, **1.3k forks**, 다수의 release를 가진 오픈소스 데스크톱 월페이퍼 도구다.  
  Source: [Lively GitHub](https://github.com/rocksdanister/lively)
- **r/Rainmeter**는 외부 통계 기준 **321k members** 규모다.  
  Source: [GummySearch - r/Rainmeter](https://gummysearch.com/r/Rainmeter/)
- **Win10 Widgets**는 Rainmeter 기반 위젯 도구로, “프로그래밍 없이도 works”라는 가치 제안을 전면에 둔다.  
  Source: [Win10 Widgets](https://win10widgets.com/)

이건 Pointint가 나중에 커서를 넘어 바탕화면, 위젯, 펫, 테마팩으로 확장할 명분이 충분하다는 뜻이다.

### 2.3 데스크톱 펫/데스크톱 버디도 다시 살아나고 있다

데스크톱 펫은 단순 향수 영역이 아니라, 최근 다시 트렌드로 묶여 보이는 신호가 있다.

- PC Gamer는 2025년 말 기준으로 **“desktop pet renaissance”**를 직접 언급한다.  
  Source: [PC Gamer - desktop pet renaissance](https://www.pcgamer.com/games/the-most-surprising-cozy-game-trend-this-year-was-the-desktop-pet-renaissance/)
- 또 다른 PC Gamer 기사에서는 **Desktop Mate**의 Sanrio DLC 캐릭터가 **각각 $14.99**에 판매된다고 소개한다.  
  Source: [PC Gamer - Desktop Mate Sanrio DLC](https://www.pcgamer.com/games/if-youve-ever-thought-that-your-desktop-is-a-bit-lonely-desktop-mate-has-just-announced-a-sanrio-dlc-that-is-so-adorable-it-might-make-my-heart-burst/)

즉, Pointint가 생각하는 “데스크톱 펫 + 공식 콘텐츠”는 너무 이른 상상이 아니라, 실제 시장 흐름과 맞닿아 있다.

## 3. 유료화 신호

무료 커서 문화가 강한 건 맞지만, **돈을 내는 사용자도 존재한다**는 신호는 분명하다.

### Etsy 신호

- Etsy의 `cursor_packs` 카테고리는 **954+ items** 규모다.  
  Source: [Etsy - Cursor Packs](https://www.etsy.com/market/cursor_packs)
- 여러 리스트가 **$2~$12**대에 형성돼 있다.
- 리뷰가 붙는 상품도 존재한다. 예:
  - **Cat Paw Mouse Cursor Set**: **527 reviews**, **$7.00**  
  - **17x Stardew Valley Cursors**: **593 reviews**, **$7.64**  
  - **Kawaii Pixel Art Animated Mouse Cursors Pack**: **$12.03**, 5-star 표시  
  Sources: [Etsy market page](https://www.etsy.com/market/cursor_packs), [Kawaii listing](https://www.etsy.com/listing/1673294578/kawaii-pixel-art-animated-mouse-cursors)
- 17종 전체 세트를 아예 상품 가치로 전면에 내세우는 사례도 보인다.  
  Source: [Pastel Bundle - all 17 states](https://www.etsy.com/listing/1806468014/pastel-bundle-minimalist-cursors-full)

### 해석

- 커서 자체는 무료 문화가 강하다
- 하지만 **“예쁘고 완성된 세트”**, **“설치가 쉬운 상품”**, **“팬덤/IP 감성”**에는 지불 의사가 생긴다
- Pointint는 여기서 “파일 판매”보다 **완성도, 편의, 생성/편집 경험**으로 차별화하는 편이 유리하다

## 4. Pointint가 실제로 파고들 수 있는 빈틈

### 4.1 기존 강자의 중심은 배포/적용 레이어다

현재 강한 플레이어들은 주로 아래에 몰려 있다.

- 다운로드 라이브러리
- 브라우저 확장
- 설치형 앱

예를 들어:

- **Custom Cursor**는 Windows 앱 설치 경로를 안내하고, 자신만의 커서를 추가하는 흐름도 앱 중심으로 설계되어 있다.  
  Sources: [How to install Custom Cursor for Windows](https://custom-cursor.com/en/how-to-use/windows/how-to-install-custom-cursor-for-windows-10-app), [How to add your own cursors](https://custom-cursor.com/en/how-to-use/windows/how-to-add-your-own-cursors-to-custom-cursor-for-windows-app)
- **Sweezy**는 이미지 업로드 커서 기능이 있지만 기본은 Chrome 확장이고, Google 정책상 브라우저 내부 페이지에서는 동작하지 않는다.  
  Source: [Sweezy Cursors Chrome Web Store](https://chromewebstore.google.com/detail/sweezy-cursors-%E2%98%85-%E0%B9%80%E0%B8%84%E0%B8%AD%E0%B8%A3%E0%B9%8C%E0%B9%80%E0%B8%8B%E0%B8%AD/gdcfjidbchfpojnfifkgghbamkdmbdaf?hl=th)
- **Custom Cursor** 역시 “without extension”은 브라우저에서 불가능하고 Windows 앱이 필요하다고 안내한다.  
  Source: [Custom Cursor Unblocked](https://custom-cursor.com/en/custom-cursor-unblocked)

### 4.2 Pointint의 빈틈은 제작 레이어다

Pointint는 아래 질문에 답하는 쪽이 맞다.

`이미지를 올리면, 내가 실제로 쓸 수 있는 Windows 커서를 가장 쉽게 만들 수 있는가?`

이 질문에 제대로 답하는 웹 기반 제품은 아직 아주 강하게 자리 잡은 대표가 보이지 않는다.

따라서 Pointint의 초기 포지셔닝은 여전히 유효하다.

- 커서 갤러리 아님
- 브라우저 확장 아님
- 설치형 에디터 아님
- **이미지 기반 Windows 커서 제작 워크플로우**

## 5. Pointint에 대한 시사점

## 5.1 지금 검증해야 할 건 “시장 존재 여부”보다 “쐐기 가치”

시장 존재 자체는 꽤 강하게 확인된다.  
이제 중요한 질문은 이거다.

`사람들이 Pointint에서 자기 이미지를 커서로 만들고, 실제로 다운로드해서 적용할 만큼 이 흐름을 가치 있게 느끼는가?`

즉, 지금 Pointint의 핵심 검증은 카테고리 검증이 아니라 **제품 쐐기 검증**에 가깝다.

## 5.2 Pointint는 커서 라이브러리 경쟁으로 가면 안 된다

이미 콘텐츠 라이브러리 경쟁은 강자가 많다.

- 방대한 무료 라이브러리
- 팬덤/IP 커서
- 브라우저 확장형 반복 사용

Pointint가 여기서 이기려면 “더 많은 커서”가 아니라 아래를 팔아야 한다.

- 더 쉬운 제작
- 더 믿을 수 있는 적용
- 더 좋은 17종 완성도
- 더 좋은 개인화

## 5.3 장기적으로는 `모니테리어` 전략이 맞다

커서만 보면 니치다.  
하지만 아래를 합치면 이야기가 달라진다.

- 커서
- 배경화면
- 테마팩
- 데스크톱 펫

Wallpaper Engine, Lively, Rainmeter, Shimeji, Desktop Mate 같은 인접 신호를 보면, Pointint의 장기 방향은 충분히 설득력이 있다.

## 6. 권장 시장 검증 실험

이번 조사 결과를 바탕으로, Pointint가 바로 실행할 가치가 있는 실험은 아래 세 가지다.

### 6.1 Concierge Test

`이미지 보내주면 내가 커서 3종 세트로 만들어드립니다`

- 채널: Reddit, X, 한국 커뮤니티
- 목표: 실제 요청 수와 요청 이미지 유형 파악
- 확인할 것: 사람들이 “남이 만든 예쁜 커서”보다 “내 이미지를 커서로 바꾸는 것”에 반응하는지

### 6.2 Landing Test

한 장짜리 랜딩에서 아래를 테스트한다.

- Hero: `Your Point, Your Tint.`
- Product line: `이미지를 나만의 Windows 커서로 바꿔보세요.`
- CTA: 이메일 대기열 또는 샘플 요청

이 테스트는 “브랜드 감각 + 제품 설명” 조합의 반응을 본다.

### 6.3 Pricing Signal Test

초기에는 직접 결제보다 아래를 본다.

- 17종 전체 세트에 대한 관심
- AI 생성 `하루 3회` 수준의 매력
- 공식 테마팩 / 펫에 대한 정서적 반응

즉, 유료화는 바로 가격이 아니라 **무엇에 돈을 낼지**를 먼저 검증해야 한다.

---

## 7. 최종 판단

### Verdict

**Go, but narrow.**

시장 자체는 충분히 존재한다.  
다만 Pointint는 “커서 꾸미기 시장” 전체를 잡으려 하기보다, 아래 쐐기로 들어가야 한다.

`내 이미지를 실제로 쓸 수 있는 Windows 커서로 만드는 가장 쉬운 제작 워크플로우`

그리고 그다음에야 아래로 넓히는 것이 맞다.

- 17종 전체 세트
- AI 보조 / 생성
- 공식 콘텐츠
- 테마팩
- 데스크톱 펫

즉, **지금은 시장이 없어서 걱정할 단계가 아니라, 올바른 진입점을 선택해야 하는 단계**다.
