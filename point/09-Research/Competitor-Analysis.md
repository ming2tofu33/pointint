## 카테고리별 경쟁사

### 1. 콘텐츠 라이브러리형 (다운로드 중심)

| 사이트 | 장점 | 단점 |
|---|---|---|
| Cursors-4U | 방대한 데이터베이스 | 디자인 노후화, 광고 과다, 고해상도 미지원 |
| VSThemes | 고퀄리티 테마, 깔끔한 미리보기 | 단순 다운로드 사이트, 제작 기능 전무 |
| DeviantArt | 창의적이고 독특한 디자인 | 적용법 복잡, 검색 불편 |
| RW-Designer | 기술적 정보와 포럼 활성화 | 1990년대 UI, 초보자 진입 어려움 |

### 2. 브라우저 확장 프로그램형

| 사이트 | 장점 | 단점 |
|---|---|---|
| Custom Cursor | 세련된 UI, 캐릭터 IP 협업 | 브라우저 전용, OS 전체 미적용 |
| Sweezy Cursors | 트렌디한 애니메이션, 접근성 | 브라우저 한정, .cur/.ani 확보 어려움 |

### 3. 제작 및 편집 도구형

| 사이트 | 장점 | 단점 |
|---|---|---|
| Cursor.cc | 웹에서 즉석 픽셀 제작 | 기능 빈약, 고해상도/AI 없음 |
| Bibata Cursor | 오픈소스 완성도 높음 | 진입장벽 매우 높음 |
| MS Store App | 시스템 커서 연동 | 앱 설치 필요, AI/웹 실시간성 부족 |

## Pointint가 채우는 빈틈

- 이미지 기반 진입 (vs 픽셀 노가다)
- OS 전체 적용 (vs 브라우저 한정)
- 시뮬레이션 검증 (vs 다운로드 후 확인)
- 웹 기반 (vs 설치형)
- AI 확장 (vs 수작업)
