# Pointint Social Content

## Context

### 제품 특성

커서는 **시각적 제품**이다. 결과물 자체가 콘텐츠가 된다. 제작 과정(before/after), 시뮬레이션 영상, 적용 결과가 모두 소셜 소재다.

### 브랜드 톤

- 감성적이되 말이 적다
- 기능 나열 ✗, 느낌 전달 ✓
- 짧은 문장, 적은 단어, 충분한 여백
- 메인 라인: `Your Point, Your Tint.`

### 타깃별 콘텐츠 접점

| 세그먼트 | 주요 채널 | 콘텐츠 훅 |
|---|---|---|
| 일반 사무직 | Twitter/X, LinkedIn | "매일 보는 커서, 한번 바꿔볼까?" |
| 커서 커뮤니티 | DeviantArt, Reddit, Discord | Hotspot/시뮬레이션 기능 시연 |
| Z세대 | TikTok, Instagram Reels | 제작 과정 타임랩스, 커스터마이징 ASMR |

## Workflow

### 1. 시스템 스킬 체이닝

`social-content` 시스템 스킬 호출 → Pointint 톤/타깃 맥락 적용

### 2. 콘텐츠 유형별 템플릿

**A. 결과물 쇼케이스**
- 형식: 이미지/짧은 영상
- 구성: 원본 이미지 → 완성 커서 → 실제 사용 장면
- 카피: 1줄 (느낌 전달)
- CTA: 링크 또는 "직접 만들어보세요"

**B. 제작 과정 (Process)**
- 형식: 타임랩스/캐러셀
- 구성: 업로드 → 편집 → Hotspot → 시뮬레이션 → 완성
- 카피: 단계별 짧은 코멘트
- 목적: "이렇게 쉽다"는 인상

**C. Before/After**
- 형식: 슬라이드/영상
- 구성: 기본 커서 → 커스텀 커서 적용된 데스크톱
- 카피: 대비가 느껴지는 한 줄

**D. 커뮤니티 참여**
- 형식: 질문/투표/챌린지
- 예: "당신의 커서 스타일은?" / "이 커서 어때요?"
- 목적: 참여 유도 + 바이럴

### 3. 플랫폼별 톤 조정

| 플랫폼 | 톤 | 형식 | 해시태그 방향 |
|---|---|---|---|
| Twitter/X | 짧고 위트 | 이미지+1줄 | #커서 #커스터마이징 #Pointint |
| Instagram | 비주얼 중심 | 캐러셀/릴스 | #데스크셋업 #모니터꾸미기 |
| TikTok | 캐주얼, 빠름 | 15-30초 영상 | #커서바꾸기 #데스크톱꾸미기 |
| Reddit | 정보+가치 | 텍스트+이미지 | 커뮤니티 규칙 준수 |
