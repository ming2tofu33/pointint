# Pointint Marketing Strategy

## Context

### 제품 정의

Pointint는 이미지를 Windows 커서로 만드는 가장 쉬운 제작 워크플로우다. 다운로드 사이트도, 단순 편집기도 아닌 **제작 워크플로우**로 포지셔닝한다.

### 브랜드 톤

- 메인 라인: `Your Point, Your Tint.`
- 감성적이되 말이 적다. 여운은 남기되 설명하지 않는다
- 기능을 나열하기보다 느낌을 전달한다
- 짧은 문장, 적은 단어, 충분한 여백

### 타깃 세그먼트 (우선순위순)

1. **일반 사무직** — "매일 보는 커서를 한번 바꿔볼까?" (쉬움 검증)
2. **기존 커서 커뮤니티** — DeviantArt, Custom Cursor 유저 (품질 검증)
3. **Z세대 커스터마이징층** — "내 데스크톱까지 나답게" (감성/공유 검증)

### 차별화 포인트

- 다운로드가 아니라 **제작**
- 이미지 한 장에서 출발, 별도 설치 없음
- Hotspot + 시뮬레이션 전면 배치
- 비회원도 강한 무료 체험

## Workflow

마케팅 관련 요청이 들어오면:

### 1. 시스템 스킬 체이닝

`marketing-ideas` 스킬 → Pointint 맥락 적용
`marketing-psychology` 스킬 → 커서 제품 특화 심리 모델 적용

### 2. 채널 우선순위 평가

Pointint의 제품 특성상 효과적인 채널 순서:

| 우선순위 | 채널 | 이유 |
|---|---|---|
| 1 | 시각적 소셜 (Twitter/X, TikTok, Instagram) | 커서는 시각적 제품 — 결과물 자체가 콘텐츠 |
| 2 | 커서/커스터마이징 커뮤니티 | DeviantArt, Reddit r/cursors, Custom Cursor 포럼 |
| 3 | 바이럴 루프 (공유 기능) | "내가 만든 커서" 자체가 공유 동기 |
| 4 | SEO/콘텐츠 | "Windows 커서 바꾸기", "커스텀 커서 만들기" 검색 |
| 5 | 유료 광고 | Phase 2 이후 |

### 3. Pointint 특화 심리 모델

마케팅 메시지 설계 시 활용:

- **IKEA 효과:** 직접 만든 커서에 더 높은 가치를 느낀다 → "나만의 커서" 강조
- **Endowment Effect:** 시뮬레이션에서 써본 커서를 잃고 싶지 않다 → 저장 = 가입 트리거
- **Zero Price Effect:** Guest 경험이 강하면 입소문이 폭발한다 → 무료 가치 강조
- **Peak-End Rule:** 시뮬레이션(peak) + 다운로드(end) 경험이 전체 인상을 결정한다

### 4. 메시지 프레이밍 규칙

Pointint 톤에 맞는 마케팅 메시지:

**DO:**
- 느낌을 전달한다 ("당신의 화면, 당신의 분위기")
- 짧고 여백이 있다
- 결과물(커서)을 보여준다
- 제작 과정의 즐거움을 전달한다

**DON'T:**
- 기능 목록을 나열하지 않는다
- 기술 용어를 전면에 내세우지 않는다
- 경쟁사를 직접 언급하지 않는다
- 설명이 길어지면 안 된다
