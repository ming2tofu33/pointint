# Pointint Canvas Design

## Context

### 브랜드 비주얼 방향

- `Your Point, Your Tint.` — 이름이 곧 비주얼 방향
- 여백 넉넉, 요소 최소, 결과물(커서) 중심
- 감성적이되 말이 적다
- 색은 분위기를 만든다

### 용도별 에셋

| 에셋 | 사이즈 | 용도 |
|---|---|---|
| OG Image | 1200×630 | 링크 공유 시 미리보기 |
| 소셜 포스트 (정사각) | 1080×1080 | Instagram, Twitter |
| 소셜 포스트 (세로) | 1080×1350 | Instagram Feed |
| 배너 | 1920×480 | 트위터 헤더, 사이트 배너 |
| 썸네일 | 400×300 | 마켓플레이스 커서 썸네일 |

---

## Workflow

### 1. 시스템 스킬 체이닝

`canvas-design` 시스템 스킬 호출 → Pointint 브랜드 톤 적용

### 2. 디자인 가이드

**DO:**
- 커서 결과물을 주인공으로 배치
- 충분한 여백
- 브랜드 라인은 작고 우아하게
- 배경은 커서를 돋보이게

**DON'T:**
- 텍스트 과다
- 복잡한 일러스트
- 네온/글리치 등 과한 효과
- 기능 목록 나열

### 3. AI 이미지 활용 시

`ai-image-generation` 시스템 스킬 → 마케팅 비주얼 생성
- 프롬프트에 Pointint 톤 반영
- 커서가 화면 위에서 사용되는 감성적 장면
- 미니멀하고 여백 있는 구성

### 4. 출력 형식

```markdown
## 에셋 제안: [제목]

### 용도
[OG/소셜/배너/기타]

### 사이즈
[px]

### 컨셉
[시각 방향 설명]

### 구성 요소
- 메인: [커서 결과물 등]
- 텍스트: [카피]
- 배경: [색/텍스처]
```

### 5. 저장 위치

- 완성 에셋: `docs/assets/` 또는 프로젝트 `public/`
- 디자인 가이드: `point/11-Design/`

## Constraints

- 색상 팔레트 미확정 시 → 중립 톤으로 진행
- 저작권 주의 — AI 생성 이미지는 라이선스 확인
