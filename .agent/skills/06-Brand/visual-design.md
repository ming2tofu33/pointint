# Pointint Visual Design

## Context

### 브랜드 시각 정체성

- `Your Point, Your Tint.` — 이름 자체가 비주얼 방향을 정의
- Point = 정확한 지점, 시선의 중심
- Tint = 분위기, 뉘앙스, 색채
- 감성적이되 말이 적다. 여운은 남기되 설명하지 않는다

### Pointint는 / Pointint는 아니다

**비주얼적으로 Pointint는:**
- 여백이 넉넉하다
- 색은 분위기를 만든다 (기능 구분이 아님)
- 깔끔하되 차갑지 않다
- 커서 결과물이 비주얼의 중심이다

**비주얼적으로 Pointint는 아니다:**
- 갤러리처럼 보이면 안 된다
- 기술 도구처럼 보이면 안 된다
- 색이 많거나 요소가 빽빽하면 안 된다

## Workflow

```

```

### 2. 디자인 결정 시 참고 프레임

**색상 팔레트:**
- 아직 미확정 (스프린트 TODO)
- 결정 시 고려: Tint 개념과의 연결, 커서 결과물을 돋보이게 하는 배경
- 기능 색보다 무드 색이 우선

**타이포그래피:**
- 짧은 문장, 적은 단어 → 타이포가 여백 안에서 돋보여야 함
- 본문은 읽기 쉽고, 헤드라인은 브랜드 느낌

**레이아웃:**
- 커서 제작 흐름이 화면의 중심
- 불필요한 네비게이션 최소화
- 모바일은 후순위 (커서 = 데스크톱 제품)
