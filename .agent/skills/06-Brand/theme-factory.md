# Pointint Theme Factory

## Context

### Pointint 브랜드 테마 정의

**톤:** 감성적이되 말이 적다. 여백은 남기되 설명하지 않는다.

**색상 방향:** (정확한 팔레트 미확정)
- Primary: 따뜻한 뉘앙스의 뉴트럴
- Accent: 감성적 포인트 컬러 (Tint를 표현)
- Background: 밝고 여백 충분
- Text: 진한 뉴트럴, 계층 분명

**타이포:**
- 헤드라인: 깔끔한 산세리프, 미디엄/세미볼드
- 본문: 읽기 쉬운 레귤러, 넉넉한 행간

**레이아웃 원칙:**
- 여백 > 요소
- 한 화면에 한 메시지
- 커서 결과물이 항상 주인공

## Workflow

### 1. 시스템 스킬 체이닝

`theme-factory` 시스템 스킬 호출 → Pointint 브랜드 테마 적용

### 2. 아티팩트별 적용

| 아티팩트 | 적용 대상 | 핵심 |
|---|---|---|
| 피치덱 (.pptx) | 슬라이드 배경, 폰트, 색상 | 여백 넉넉, 1슬라이드 1메시지 |
| 보고서 (.docx) | 헤더, 본문 스타일 | 깔끔한 계층 구조 |
| 스프레드시트 (.xlsx) | 헤더 색상, 테이블 스타일 | 가독성 우선 |
| HTML 랜딩 | CSS 변수, 컴포넌트 | 디자인 토큰 연동 |
| 이미지 에셋 | 색상, 구성 | 브랜드 톤 일관성 |

### 3. 테마 체크리스트

아티팩트 완성 후 반드시 확인:

- [ ] Pointint 색상 방향에 맞는가?
- [ ] 여백이 충분한가?
- [ ] 텍스트가 짧고 명확한가?
- [ ] 커서 결과물/제품이 중심에 있는가?
- [ ] 갤러리/기술 도구처럼 보이지 않는가?

## Constraints

- 색상 팔레트 확정 전까지는 방향성만 적용 (구체적 hex는 유보)
- 과한 그라디언트, 그림자, 장식 요소 금지
- 일관성 > 화려함
