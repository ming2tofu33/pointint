# 비즈니스 상세

## 1. 현재 기준 티어 구조

Pointint의 비즈니스 구조는 “제작 자체는 넓게 열고, 더 깊은 개인화와 판매 권한에서 돈을 받는 방식”이 가장 자연스럽다.

### 티어별 권한 경계

| 기능 | Guest | Free | Lite | Pro |
|---|---|---|---|---|
| 업로드 기반 커서 제작 (.cur) | O | O | O | O |
| 기본 전처리 / 배경 정리 | O | O | O | O |
| 편집 / Hotspot / 3종 시뮬레이션 | O | O | O | O |
| 3종 다운로드 | O | O | O | O |
| 프로젝트 저장 / 재수정 | X | O | O | O |
| 17종 전체 커서팩 다운로드 | X | O | O | O |
| AI 보조 편집 | X | X | O | O |
| AI 생성 | X | X | 하루 3회 | 더 높은 한도 |
| 마켓플레이스 판매 등록 | X | X | O | O |
| 내부 재화 지갑 사용 | X | X | O | O |
| 노출 우대 / 수수료 우대 | X | X | X | O |

### 경계의 원칙

- 제작 자체는 막지 않는다
- Guest -> Free 전환은 저장과 17종 전체 세트가 만든다
- Free -> Lite 전환은 AI와 판매 권한이 만든다
- Lite -> Pro 전환은 더 많은 AI와 더 좋은 판매 조건이 만든다

---

## 2. 가격대 방향

아직 확정 가격은 아니지만, 현재 감각 기준의 방향은 아래가 적절하다.

| 티어 | 월 가격 가안 | 포지션 |
|---|---|---|
| Lite | ₩3,900 전후 | 부담 없이 결제하는 첫 유료층 |
| Pro | ₩9,900 전후 | 진지한 크리에이터 / 상위 유저층 |

### 가격 원칙

- Lite는 “가볍게 시작할 수 있는 가격”이어야 한다
- Pro는 “더 많은 결과물”보다 “더 깊은 운영 능력”을 팔아야 한다

---

## 3. 마켓플레이스 구조

마켓플레이스는 MVP가 아니라, 제품 가치가 먼저 검증된 후 열어야 한다.

### 초기 운영

- 판매 등록은 Lite 이상
- 초기 보상은 현금 정산이 아니라 내부 재화
- 권리 확인 동의 후 등록
- 정책 위반 콘텐츠는 제거

### 수수료 방향

| 티어 | 초기 가안 |
|---|---|
| Lite | 20% |
| Pro | 10% |

이 수치는 추후 조정 가능하지만, 초기에는 “크리에이터 유인”을 해치지 않는 것이 중요하다.

---

## 4. 내부 재화

내부 재화는 Pointint 생태계를 돌리는 핵심 장치다.

### 초기 역할

- 추가 AI 사용
- AI 보조 / 생성 기능 접근

### 중기 역할

- 공식 테마팩 구매
- 데스크톱 펫 팩 구매
- 시즌 드롭 구매
- 마켓 노출 강화

### 핵심 원리

- AI는 재화의 바닥 수요를 만든다
- 콘텐츠는 재화의 감정적 가치를 만든다

---

## 5. 상품 확장 순서

Pointint의 상품은 한 번에 넓어지지 않는다.

1. 커서 제작
2. AI 보조 / 생성
3. 마켓플레이스
4. 공식 콘텐츠
5. 테마팩 경제

### 후속 상품 예시

- 공식 테마팩
- 시즌 컬렉션
- 데스크톱 펫 팩
- 이후 `.ani` 애니메이션 커서 확장

---

## 6. 유료화 한 줄

제작은 넓게 열고, AI로 전환시키고, 콘텐츠와 판매 구조로 생태계를 만든다.
