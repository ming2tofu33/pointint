# 경쟁 사이트

### Gemini said

제시해주신 사이트들은 마우스 커서 업계의 '조상님'부터 '현역 스타'까지 모두 모인 아주 좋은 벤치마킹 리스트입니다. 하지만 냉정하게 분석해 보면, 대부분 **2000년대 기술에 머물러 있거나 특정 환경(브라우저)에 갇혀 있는** 경우가 많습니다.

AI 엔지니어이신 사용자님의 **'Pointint'**가 이들과 비교했을 때 갖는 압도적인 차별점을 **4가지 카테고리**로 분류해 정리해 드립니다.

---

## **📊 경쟁 서비스 vs Pointint 비교 분석**

### **1. 제작 방식의 혁신: "노동에서 AI로"**

- **기존 (Cursor.cc, RW-Designer):** 사용자가 직접 픽셀을 하나하나 찍거나, 미리 완성된 파일을 다운로드만 해야 합니다. 배경 제거 기능이 없거나 매우 조잡합니다.
- **Pointint:** 일반인도 평범한 사진(JPG/PNG)만 올리면 **AI가 자동으로 배경을 날리고(rembg)** 커서 규격으로 변환해 줍니다. "만드는 재미"와 "편의성"을 동시에 잡았습니다.

### **2. 가짜 커서 vs 진짜 커서 (The "Real" OS Integration)**

- **기존 (Custom Cursor, Sweezy):** 대부분 **크롬 확장 프로그램** 방식입니다. 브라우저 안에서는 화려하지만, 브라우저를 끄거나 폴더를 열면 다시 기본 화살표로 돌아오는 **'반쪽짜리'** 경험입니다.
- **Pointint:** 윈도우 표준인 `.cur`와 `.ani` 파일을 직접 생성하고, 이를 한 번에 등록하는 **`.inf` 설치 스크립트**까지 제공합니다. PC 전체 환경을 바꾸는 **'진짜' 커스터마이징**을 지향합니다.

### **3. 검증 환경의 유무: "17종 실시간 시뮬레이터"**

- **기존 (모든 사이트):** 파일을 다운로드해서 윈도우 설정창에 일일이 등록해 보기 전까지는 실제 느낌을 알 수 없습니다. 특히 애니메이션 커서(`.ani`)의 경우 속도가 적당한지 확인하기가 매우 어렵습니다.
- **Pointint:** 웹사이트 내에서 윈도우의 모든 상태(대기, 텍스트, 링크 등)를 재현한 그리드를 제공합니다. 마우스만 올려보면 **적용 후의 모습을 100% 확신**할 수 있습니다.

### **4. 기술적 현대화: "Legacy vs Modern"**

- **기존 (Cursors-4u, vsthemes):** 사이트 자체가 십수 년 전 디자인이며, 보안이 취약해 보이거나 광고가 지나치게 많습니다. 고해상도(4K/8K) 모니터 대응이 안 된 저품질 에셋이 많습니다.
- **Pointint:** 최신 웹 기술(React, FastAPI)과 깔끔한 UI를 기반으로 하며, 고해상도 안티앨리어싱 처리가 포함된 고퀄리티 파일 생성을 지원합니다.

| **비교 항목** | **기존 다운로드형 (DeviantArt 등)** | **브라우저 확장형 (Custom Cursor 등)** | **Pointint (User's Project)** |
| --- | --- | --- | --- |
| **적용 범위** | OS 전체 (수동 등록) | **브라우저 내부 한정** | **OS 전체 (자동 설치 지원)** |
| **제작 난이도** | 매우 높음 (전문 툴 필요) | 낮음 (제공된 것만 사용) | **매우 낮음 (AI 자동 변환)** |
| **애니메이션** | 지원하지만 제작 어려움 | 자체 규격 사용 | **GIF 업로드로 .ani 자동 생성** |
| **신뢰도** | 구식 사이트, 광고 많음 | 확장 프로그램 권한 요구 | **깔끔한 웹 기반 도구** |
| **미리보기** | 불가능 (적용 후 확인) | 실시간 (브라우저 내) | **17종 시스템 상태 실시간 시뮬레이션** |

## 💡 결론: 사용자님의 프로젝트가 갖는 필승 전략

사용자님의 프로젝트는 **"제작의 즐거움(AI)"**&과 **"확실한 결과물(17종 시뮬레이터)"**, 그리고 **"완벽한 적용(.inf 패키징)"**이라는 세 마리 토끼를 다 잡고 있습니다.

특히 경쟁사 중 `cursor.cc`는 너무 낡았고, `Custom Cursor`는 브라우저용이라는 한계가 명확하기 때문에, **"윈도우 전체를 꾸미고 싶은 MZ세대와 데스크테리어족"**에게는 사용자님의 서비스가 유일한 대안이 될 것입니다.

## 💡 종합 비교: Pointint가 이들을 압도하는 지점

위의 사이트들을 종합해 보면, **"쉽게 만들면서(AI), 모든 곳에 적용되고(OS 전체), 미리 확인(17종 시뮬레이터)까지 가능한"** 서비스는 단 하나도 없습니다.

### 🚀 우리 프로젝트만의 킬러 콘텐츠

1. **AI 배경 제거 (vs Cursor.cc):** 픽셀을 찍는 노가다 대신, 사진 한 장으로 끝냅니다.
2. **OS 전체 적용 (vs Custom Cursor):** 브라우저라는 감옥을 탈출하여 윈도우 전체를 바꿉니다.
3. **17종 실시간 시뮬레이션 (vs 모든 경쟁사):** 파일을 다운로드하기 전에 **"텍스트를 쓸 때, 링크를 누를 때"**의 느낌을 웹에서 미리 겪어보게 합니다.
4. **.inf 원클릭 설치 (vs DeviantArt):** 복잡한 수동 등록 절차를 자동화 스크립트로 해결합니다.
