# Cursor Explore Object Prompts

These prompts are tuned for Pixellab-generated transparent-background pixel objects that can later be adapted into cursor assets.

## Shared Direction

- Fill most of a square frame.
- Read clearly at small size.
- Pixel-inspired object image, not an app icon.
- Simple silhouette, strong highlights, restrained internal lines.
- Transparent background required.
